# reliable-sources
Project to return a list of reliable on-the-scene sources from Twitter.

## Set up and Installation ##
Set up virtualenv

Run `pip install -r requirements.txt`

## Usage ##
`python main.py <Location>`

Example: `python main.py Toronto`
